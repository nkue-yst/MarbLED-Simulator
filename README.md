# MatrixLED-Simulator
実行時に `-d` オプションを付けてアプリ動作中のデバイスのIPアドレスを指定することで、別マシンで動作中のアプリケーションの様子も確認することができる。
また、シミュレータウィンドウ内をマウスクリックすると、その座標に対するタッチイベントを発生させられる。

# 実行例
- 192.168.1.2が振られたデバイス上でアプリ起動している場合
```
$ ./MatrixLED-Simulator -d 192.168.1.2
```
